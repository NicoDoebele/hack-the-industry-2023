# Hackathon 2023

## Get started

### Clone the project
```bash
git clone https://github.com/NicoDoebele/hackathon.git
```

### Change Directory
```bash
cd hackathon
```

## Run Frontend and Backend

Run Backend on Port 3001
```bash
cd backend
npm install
npm start
```

Run Frontend on Port 3000
```bash
cd ..
cd frontend
npm install
npm start
```
